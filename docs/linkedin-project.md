## Project title

Energy Market Data Lake

## One-line version

Built a cost-optimized AWS data lake for UK and EU energy market data using Lambda, S3, Glue, Athena, EventBridge, and a React dashboard for presentation-ready analytics.

## Profile-ready description

Designed and deployed an AWS-based energy market data lake that ingests UK and EU electricity and gas datasets, stores raw data in S3, transforms it into partitioned Parquet with Glue, and analyzes it through Athena. Added a React dashboard to present market trends, imbalance pricing, hedge coverage, and portfolio risk in a portfolio-ready format. Focused on cost control through lifecycle policies, partitioning, and lightweight scheduled ingestion.
